# Instrucciones

```bash
npm run start
```
o

```bash
npm run dev
```

Se estara creando usuario nuevo al igual como eliminar, actualizar y obtener los cursos.

## Rutas
- GET    /api/usuarios
- GET    /api/usuarios/:id_usuario
- POST   /api/usuarios
- UPDATE /api/usuarios/:id_usuario
- DELETE /api/usuarios/:id_usuario

| id_usuario | nombre | email    | password |
|------------|--------|----------|----------|
