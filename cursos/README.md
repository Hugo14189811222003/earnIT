# Instrucciones

```bash
npm run start
```
o

```bash
npm run dev
```

Se estara creando cursos nuevo al igual como eliminar, actualizar y obtener los cursos.

## Rutas
GET    /api/cursos
GET    /api/cursos/:id_curso
POST   /api/cursos
UPDATE /api/cursos/:id_curso
DELETE /api/cursos/:id_curso

| id_curso | titulo | descripción | duración |
|-----------|-------|------------|------------|
